# day-trip

Plan perfect day trips — no overnight stay needed. Optimizes a full day of activities within a city or to nearby attractions, returning by evening.

## Overview

Plan perfect day trips — no overnight stay needed. Optimizes a full day of activities within a city or to nearby attractions, returning by evening. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install day-trip
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/day-trip
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Plan perfect day trips — no overnight stay needed. Optimizes a full day of activities within a city or to nearby attractions, returning by evening.

**Command type:** Dual-command
